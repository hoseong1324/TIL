### 문자열 치환



String str = "aaaabbbbcccabc";

str.replace("abc","깃");

/// 출력값 ===  aaaabbbbccc깃

str.replaceAll("[abc]","깃");

/// 출력값 ==== 깃깃깃깃깃깃깃깃깃깃깃깃깃깃



### 특수코드 치환 ( HTML Character Entity [ HTML 문자단위 ] )
```
&nbsp;
” ” : 공백(스페이스 한 칸)을 의미
 
&lt;
부등호(<)
 
&gt;
부등호(>)
 
&amp;
앰퍼샌드(&) 기호
 
&quot;
쌍따옴표(“)
 
&#035;
sharp(#)
 
&#039;
따옴표(‘)
```
**HTML 에서 특수문자를 보낼때는 특수코드로 날라오기때문에 치환하여 데이터를 저장하거나 뿌려주어야 한다 !**
